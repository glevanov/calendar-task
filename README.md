# calendar-task

A small calendar app built with Vue & Typescript.

Demo is available at [Github Pages](https://glevanov.github.io/calendar-task/).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Runs unit tests
```
yarn run test:unit
```
